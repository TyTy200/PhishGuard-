# PhishGuard-
PhishGuard Is A Phishing Detection Web Application &amp; Additional (BE) For Safer Browsing Online 
READ ME.

How to use;
- Download Visual Studio
- Download PhishGuard from depository
- install flask in terminal (pip install flask,
- if that command fails run ( python -m pip install flask

- to verify installation run: python -c 'import flask; print(flask.__version__)'

install pickle in terminal (pip install pickle)
install numpy in terminal (pip intsall numpy)
install pandas in terminal (pip install pandas)
install PIL in terminal (pip install pil)

TO LAUNCH, click run on app.py file --> start debugging --> select debug configuration --> flask --> app.py
click local host in terminal 
for browser extension, --> open chrome --> open browser extension --> turn on developer mode --> install backend --> browser extension file onto extension by dragging single (be) file. version 1.0
